# Drops

### What is a Drop?

A Drop is a piece of media on Zora.

![Drop of Zora](https://nftstorage.link/ipfs/bafybeifwdixgvmoznitkhyf5dewyx4qed6ceusii6bhluvagp3xbjfukgy)

### Getting Started

```bash
npm install onchain-magic
# or
yarn add onchain-magic
```

once you've installed `onchain-magic` you can use the `use1155Drop` hook in your app
