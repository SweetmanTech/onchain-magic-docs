# Introduction to OnchainMagic

OnchainMagic is a suite of open-source tools that allow developers to easily integrate Zora into their applications.

OnchainMagic currently provides:

- A [Leaderboard](/leaderboard)
- A [front-end SDK](https://www.npmjs.com/package/onchain-magic) for interacting with Zora
- An [mobile-first app](https://onchainmagic.xyz) for viewing and interacting with Zora

## What is Zora?

Zora helps creators upload imagination and earn ETH. Zora is unique in its approach because it is built as a hyperstructure.

Read more about Zora [here](https://zora.co).
