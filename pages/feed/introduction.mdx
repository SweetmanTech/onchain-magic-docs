# Feed

### What is the Feed?

The Feed is a social feed of the latest drops on Zora.

1. tap to play / pause videos and audio.
2. like / collect the active media.
3. leave a comment.
4. view the media on Zora.
5. title of the media.

![Feed of Zora](https://nftstorage.link/ipfs/bafybeigkrednwtnxsgszcyq4jui57whfx54fpczmww4tl3ytshpa5fcpvy)

### Getting Started

```bash
npm install onchain-magic
# or
yarn add onchain-magic
```

once you've installed `onchain-magic` you can use the `useFeed` hook in your app
